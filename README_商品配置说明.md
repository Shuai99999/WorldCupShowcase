# 商品配置说明

## 如何添加商品

### 1. 编辑商品数据文件

打开 `src/data/products.json` 文件，添加或修改商品信息。

### 2. 商品数据格式

每个商品需要包含以下字段：

```json
{
  "id": "product-001",              // 唯一ID（必需）
  "name": "商品名称",                // 商品名称（必需）
  "description": "商品详细描述",     // 商品描述（必需）
  "images": [                        // 商品图片数组（必需）
    "/products/image1.jpg",
    "/products/image2.jpg"
  ],
  "category": "球衣",                // 商品分类（可选）
  "tags": ["官方", "正品"],          // 标签数组（可选）
  "source": "淘宝"                   // 来源（可选）
}
```

### 3. 图片存放位置

- 将商品图片放在 `public/products/` 目录下
- 在 `products.json` 中使用相对路径：`/products/你的图片名.jpg`
- 支持多张图片，第一张图片会作为主图显示
- 如果图片路径不正确，会自动显示占位图

### 4. 示例

参考 `src/data/products.template.json` 文件查看完整的配置模板。

### 5. 添加商品的步骤

1. 从淘宝或1688收集商品信息（名称、描述、图片）
2. 将图片保存到 `public/products/` 目录
3. 在 `src/data/products.json` 中添加商品对象
4. 保存文件后，网站会自动更新显示新商品

### 6. 注意事项

- 图片建议尺寸：宽度 800-1200px，比例 4:3 或 3:4
- 图片格式：JPG、PNG、WebP 均可
- 商品ID必须唯一
- 如果商品有多张图片，建议将最清晰的图片放在第一张
